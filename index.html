stonk

cli tool to monitor indian stocks data.

install

npm i @anujsharma141/stonk -g

usage

-s, --search

search from stocks listed

stonk --search <keyword>

-a, --add

add stock to watchlist

stonk --add <ticker>

-r, --remove

remove stock from watchlist

stonk --remove <ticker>

-i, --info

logs quote value of a stock

stonk --info <ticker>

-w, --watch

logs the watchlist

stonk --watch

--help

show help

stonk --help
