## :rocket: Install

```bash
npm i @anujsharma141/lyrics -g
```


## :pushpin: Usage

```bash
lyrics <song> <artist>
```

## :wrench: Development
Run the following commands in terminal to setup the project on your local machine.

```bash 
git clone https://github.com/AnujSharma141/lyrics.git
cd lyrics
npm install
npm start
```
